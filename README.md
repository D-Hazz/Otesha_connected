# Otesha Connecté

Otesha Connecté est un projet de système intelligent connecté (IoT) conçu pour la surveillance et le contrôle en temps réel d’un environnement physique. Il s’appuie sur l’intégration de capteurs, d’un microcontrôleur et d’une interface logicielle afin de collecter, analyser et exploiter des données de manière fiable.

## Objectifs du projet

- Mettre en œuvre une solution IoT fonctionnelle et modulaire
- Assurer la collecte et la transmission de données en temps réel
- Visualiser les données via une interface simple et exploitable
- Concevoir une architecture adaptée aux contextes à ressources limitées
- Servir de base pédagogique et expérimentale en systèmes intelligents

## Architecture générale

Le système est structuré autour de trois couches principales :

1. **Couche matérielle**
   - Capteurs (température, humidité ou autres selon le cas d’usage)
   - Microcontrôleur pour l’acquisition et le prétraitement des données

2. **Couche communication**
   - Transmission des données vers le système central
   - Protocoles légers adaptés aux contraintes matérielles

3. **Couche logicielle**
   - Backend pour la réception et la gestion des données
   - Interface utilisateur pour la visualisation et le suivi en temps réel

## Technologies utilisées

- Microcontrôleur (ex. Arduino, ESP32 ou équivalent)
- Capteurs environnementaux
- Backend (Python / Django ou équivalent)
- Frontend web (HTML, CSS, JavaScript)
- Base de données légère pour l’historisation

## Cas d’utilisation possibles

- Surveillance environnementale
- Systèmes intelligents éducatifs
- Projets IoT à faible coût
- Prototypage de solutions connectées locales

## État du projet

Le projet est en cours de développement et évoluera progressivement avec :
- l’ajout de nouveaux capteurs
- l’amélioration de l’interface utilisateur
- l’optimisation de la collecte et de l’analyse des données
